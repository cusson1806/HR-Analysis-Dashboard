# 📊 HR Analytics Dashboard (Excel)

## Overview

This project is an interactive **HR Analytics Dashboard** developed in **Microsoft Excel** to analyze employee demographics, workforce distribution, and attrition trends. The dashboard transforms raw HR data into meaningful insights through KPIs, Pivot Tables, Pivot Charts, and Excel formulas, enabling HR professionals and business leaders to make data-driven workforce decisions.

The project focuses on identifying key factors influencing employee attrition, including department, job role, gender, age, salary, overtime, and years spent at the company.

---

## Objectives

* Analyze employee attrition patterns.
* Monitor workforce demographics.
* Compare attrition across departments and job roles.
* Evaluate the impact of overtime on employee turnover.
* Examine employee age and salary distributions.
* Present key HR metrics through an interactive dashboard.

---

## Dashboard KPIs

The dashboard provides the following key performance indicators (KPIs):

* Total Employees
* Employees Left
* Attrition Rate
* Average Age
* Average Monthly Income
